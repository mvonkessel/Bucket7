Regeln & Value Score
Value Score — Preis-Leistung pro Bucket, PNI-kaufkraftnormiert nach Weber-Fechner.
V = Enorm ÷ (β₁ · ln(1 + Budgetmid) / ln(10001) + β₂ · Scary/10 + ε)

Enorm = Excitement ÷ 10
Budgetmid = (Min + Max) / 2 · logarithmisch (Weber-Fechner)
β₁ = 0.65 · β₂ = 0.35 · ε = 0.1
Budget-Range: Min–Max Budget pro Land. Value Score nutzt den Midpoint. Breiter Range = flexibles Budget.
